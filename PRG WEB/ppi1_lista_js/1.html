O que é JavaScript?

O JavaScript foi criado em 1995 por Brendan Eich (na Netscape) com o objetivo de tornar as páginas web dinâmicas. Antes dele, os sites eram estáticos, sem interatividade. A ideia era permitir que scripts rodassem diretamente no navegador, manipulando elementos da página sem recarregar (ex.: validação de formulários, animações).

A execução no navegador ocorre no computador do usuário, e é passível de manipulação do DOM, HTML e CSS, além de ter APIs como document e alert. No servidor ele apenas lida com requisições em uma máquina remota, sem acesso ao DOM tem APIs de sistema.

Sandbox é um ambiente isolado onde o JavaScript executa no navegador, com restrições de segurança para evitar ações maliciosas. Ele não pode acessar o sistema de arquivos do usuário ou fazer requisições a outros domínios sem permissão. É como uma caixa de areia que protege a máquina do usuário de códigos perigosos.

Objetos globais
* document: Representa o DOM e permite manipular elementos
* console: Usado para DEPURAÇÃO, ou seja, exibição de mensagens no console do navegador
* alert: Exibe uma caixa de diálogo pop-up

Estes são ferramentas essenciais para interagir com a página, registrar mensagens e exibir alertas. São globais, não precisam ser declarados e só funcionam no navegador.
